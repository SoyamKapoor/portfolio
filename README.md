# 🌐 Soyam Kapoor — Personal Portfolio

> A fully custom-built personal portfolio website showcasing my journey as a Data Scientist, Machine Learning Engineer, and final-year Computer Systems Engineering student.

**Live Site →** [your-netlify-link.netlify.app](https://soyamkapoor.github.io/portfolio/)

---

## ✨ Features

- **Three.js WebGL** animated particle network background
- **Lenis** smooth scrolling
- Custom cursor with magnetic button effects
- Scroll-reveal animations and text scramble effects
- Interactive **Skill Radar Chart** (Chart.js)
- Light / Dark theme toggle
- 20+ project cards with direct GitHub links
- Experience timeline with internships and roles
- IEEE research publication section
- 9 verified certifications with credential links
- Contact form powered by **Formspree**
- Fully responsive across all screen sizes
- Performance optimised — deferred JS, GPU-efficient Three.js

## 📁 Project Structure

```
portfolio/
├── index.html          # Main HTML — all sections
├── css/
│   ├── base.css        # Tokens, reset, loader, cursor
│   ├── layout.css      # Nav, hero, modal, footer
│   └── sections.css    # About, skills, projects, contact
├── js/
│   ├── three-bg.js     # WebGL particle network
│   ├── animations.js   # Cursor, scroll, tilt, typing
│   └── main.js         # Loader, theme, radar, modals
└── assets/
    ├── img/            # Profile photo
    └── CV_SOYAM.pdf    # Downloadable CV
```

## 🚀 Getting Started

No build tools or dependencies required. Just clone and open.

```bash
git clone https://github.com/SoyamKapoor/portfolio.git
cd portfolio
```

Then open `index.html` in your browser, or use **Live Server** in VS Code.
